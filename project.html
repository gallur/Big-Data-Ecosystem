Finding Euler Graphs

Tech Stack: Python, MRJob, Hadoop, MapReduce.

Description: An algorithm that tells us whether a given graph of size and complexity, is a Euler graph or not. In other words, it tells us whether this graph can be traversed without having to traverse an edge twice.
